# Kanye Quote Generator

iOS app built with SwiftUI that fetches quotes from online APIs and lets users guess whether the quote is from Kanye West or another author.

## Features
- Fetch quotes from web APIs
- Guess game with score system
- Save, add, edit and delete favorite quotes
- Works in portrait and landscape

## Technologies
SwiftUI, SwiftData, async/await networking

## APIs
- https://api.kanye.rest  
- https://zenquotes.io

## Run
Open the project in Xcode 16+ and run on an iOS 18+ simulator or device.
